# 📍 دليل الوصول إلى إعدادات الموقع

## ✅ تم التأكد من:

- ✔️ Backend يعمل بنجاح على http://localhost:5000
- ✔️ قاعدة البيانات متصلة
- ✔️ البيانات الافتراضية موجودة
- ✔️ API endpoint يعمل: `/api/settings/venue`

## 📝 خطوات الوصول إلى صفحة الإعدادات:

### 1. تسجيل الدخول

افتح المتصفح واذهب إلى: `http://localhost:5173/admin/login`

### 2. الوصول إلى لوحة الإدارة

بعد تسجيل الدخول، ستجد القائمة الجانبية تحتوي على:

- 📊 Overview (نظرة عامة)
- 👥 Committees (اللجان)
- 🎤 Keynote Speakers (المتحدثون)
- ✅ Participants (المشاركون)
- 📄 Reviewer Requests (طلبات التحكيم)
- 📧 Contact Messages (الرسائل)
- ⚙️ **Settings (الإعدادات)** ← **هنا!**

### 3. اضغط على Settings

في أسفل القائمة الجانبية، ستجد زر **"Settings"** أو **"الإعدادات"**

### 4. تعديل معلومات الموقع

ستشاهد نموذجاً يحتوي على:

#### معلومات باللغة الإنجليزية:

- 📍 Venue Name (English)
- 📍 Address (English)
- 📝 About (English)

#### معلومات باللغة العربية:

- 📍 اسم الموقع (بالعربية)
- 📍 العنوان (بالعربية)
- 📝 حول الموقع (بالعربية)

#### معلومات الاتصال:

- ☎️ Phone Number
- 📧 Email Address
- 🗺️ Google Maps URL

### 5. حفظ التغييرات

بعد التعديل، اضغط على **"Save Changes"** أو **"حفظ التغييرات"**

---

## 🔍 التحقق من التغييرات:

### 1. الصفحة الرئيسية

اذهب إلى: `http://localhost:5173`
ستجد اسم الموقع في قسم تفاصيل المؤتمر

### 2. صفحة الموقع

اذهب إلى: `http://localhost:5173/venue`
ستجد جميع المعلومات المحدثة:

- اسم الموقع
- العنوان
- نبذة عن الموقع
- رقم الهاتف
- البريد الإلكتروني
- رابط الخريطة

### 3. صفحة الجدول الزمني

اذهب إلى: `http://localhost:5173/schedule`
ستجد اسم الموقع يظهر في كل فعالية

## 🚀 تشغيل التطبيق:

```bash

```

## ❓ حل المشاكل:

### إذا لم تظهر صفحة الإعدادات:

1. تأكد من تسجيل الدخول بحساب Admin
2. افحص console المتصفح (F12)
3. تأكد من أن Frontend و Backend يعملان
4. امسح cache المتصفح (Ctrl+Shift+R)

### إذا لم يتم حفظ التغييرات:

1. افحص console المتصفح للأخطاء
2. تأكد من صحة البيانات المدخلة
3. تأكد من الاتصال بالإنترنت (للوصول لقاعدة البيانات)
