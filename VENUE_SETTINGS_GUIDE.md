# نظام إعدادات الموقع الديناميكي

تم إضافة نظام كامل لجعل معلومات موقع المؤتمر ديناميكية وقابلة للتحديث من لوحة الإدارة.

## ✨ المزايا

- **موقع ديناميكي**: يمكن للأدمن تغيير معلومات الموقع من لوحة التحكم
- **متعدد اللغات**: دعم كامل للغتين العربية والإنجليزية
- **تحديث تلقائي**: جميع الصفحات تعرض المعلومات المحدثة تلقائياً
- **واجهة سهلة**: نموذج بسيط وواضح في لوحة الإدارة

## 🚀 التثبيت والإعداد

### 1. تهيئة الإعدادات الافتراضية

قم بتشغيل الأمر التالي لإنشاء سجل الإعدادات الافتراضي في قاعدة البيانات:

```bash
cd backend
npm run init-settings
```

### 2. إعادة تشغيل الخادم

إذا كان الخادم يعمل، قم بإعادة تشغيله لتحميل التغييرات:

```bash
npm run dev
```

## 📝 كيفية الاستخدام

### للأدمن:

1. سجل الدخول إلى لوحة الإدارة
2. اختر "الإعدادات" (Settings) من القائمة الجانبية
3. قم بتعديل معلومات الموقع:
   - اسم الموقع (بالعربية والإنجليزية)
   - العنوان (بالعربية والإنجليزية)
   - نبذة عن الموقع (بالعربية والإنجليزية)
   - رقم الهاتف
   - البريد الإلكتروني
   - رابط Google Maps
4. اضغط "حفظ التغييرات"

### الصفحات المتأثرة:

المعلومات المحدثة ستظهر تلقائياً في:

- **الصفحة الرئيسية**: عرض اسم الموقع
- **صفحة الموقع**: جميع معلومات الموقع بالتفصيل
- **صفحة الجدول الزمني**: عرض اسم الموقع

## 🔧 البنية التقنية

### Backend:

- **Model**: `Settings.ts` - نموذج MongoDB للإعدادات
- **Service**: `settings.service.ts` - منطق الأعمال
- **Controller**: `settings.controller.ts` - معالجة الطلبات
- **Routes**: `settings.routes.ts` - المسارات
  - `GET /api/settings/venue` - جلب معلومات الموقع (عام)
  - `PUT /api/settings/venue` - تحديث معلومات الموقع (للأدمن فقط)

### Frontend:

- **Admin Component**: `SettingsManagement.tsx` - واجهة تعديل الإعدادات
- **API Service**: إضافة `settingsAPI` في `api.ts`
- **Updated Pages**:
  - `Venue.tsx` - جلب البيانات من API
  - `LandingPage.tsx` - جلب اسم الموقع ديناميكياً
  - `Schedule.tsx` - جلب اسم الموقع ديناميكياً

## 🔐 الأمان

- نقطة الجلب عامة للجميع (`GET /api/settings/venue`)
- نقطة التحديث محمية ومتاحة للأدمن فقط (`PUT /api/settings/venue`)
- استخدام middleware للمصادقة: `adminAuthMiddleware`

## 📊 البيانات المخزنة

```typescript
{
  venue: {
    nameEn: string; // اسم الموقع بالإنجليزية
    nameAr: string; // اسم الموقع بالعربية
    addressEn: string; // العنوان بالإنجليزية
    addressAr: string; // العنوان بالعربية
    aboutEn: string; // نبذة بالإنجليزية
    aboutAr: string; // نبذة بالعربية
    phone: string; // رقم الهاتف
    email: string; // البريد الإلكتروني
    mapUrl: string; // رابط Google Maps
  }
}
```
